# Rise Player (Electron)

This is a Display Module that is responsible for showing all of the playable content on the display.

### Run it

### Install dependencies
```
npm install
```

### Launch Electron
```
npm start
```

### Unit Test
```
npm run test
```

### Integration Test
```
npm run integration
```

### Build
```
./build.sh
```

### Deploy
```
./deploy.sh
```
