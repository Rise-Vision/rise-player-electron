# Rise Player (Electron)

This is a Display Module that is responsible for showing all of the playable content on the display.

### Run it

### Install dependencies and launch Electron
```
npm install
npm start
```

### Unit Test
```
npm run test
```

or

```
npm run test-no-coverage
```

### Integration Test
```
npm run integration
```

### Build
```
./build.sh
```

### Deploy
```
./deploy.sh
```
